# TranslatorTool

Translates text between any two languages with context-aware phrasing.

## Installation

```bash
pip install -r requirements.txt
cp .env.example .env  # add your ANTHROPIC_API_KEY
```

## Usage

```bash
python translator_tool.py
```

## Model

`claude-sonnet-4-20250514` via the Anthropic Python SDK.
